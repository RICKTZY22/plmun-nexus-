◆ Diagram 5

Project Charter

Formal authorization document outlining the project's vision, scope, and governance.

🎯 Vision

Develop a centralized, secure, web-based inventory management system replacing PLMun's manual tracking with real-time digital workflows and transparent accountability.

📋 Scope

  • Inventory CRUD with access-level filtering
  • Request workflow (Pending → Approved → Returned)
  • 4-tier RBAC (Student/Faculty/Staff/Admin)
  • Dashboard analytics + PDF/CSV export
  • Cloud deployment (Vercel + Railway)

⚠️ Constraints

  • Academic timeline (1 semester)
  • 6-person student team
  • Free-tier cloud hosting
  • No institutional IT budget
  • Open-source tools only
