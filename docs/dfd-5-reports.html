Reporting & Data Export

Sub-Process 5.3
Export to CSV
Exports inventory items or request data to CSV format. Uses the File System Access API for a native "Save As" dialog with fallback to anchor download.
🛡️ Staff / Admin
Export CSV action
5.3
Export
CSV
.csv file download
📄 File System
Format: UTF-8 with BOM (\\uFEFF) for Excel compatibility. Comma-delimited. Quoted fields.
Filename: inventory_YYYY-MM-DD.csv or requests_YYYY-MM-DD.csv
Inventory CSV Headers: Name, Category, Quantity, Status, Location, Description, Date Added
Reports CSV Headers: Request ID, Item, Requester, Quantity, Status, Date, Purpose
Sub-Process 5.4
Export to PDF
Generates a formatted PDF document with title, summary statistics, data table, and branded footer. Client-side generation using jsPDF.
🛡️ Staff / Admin
Export PDF action
5.4
Export
PDF
.pdf file download
📄 File System
Library: jsPDF + jspdf-autotable (client-side, no server round trip)
PDF Layout: Title → Generated timestamp → Summary stats → Auto-table with headers/rows → Page numbers + "PLMun Inventory Nexus" branding
Table Style: Grid theme, blue header (#2563eb), alternating row colors, 8pt font
Save Dialog: Uses showSaveFilePicker() API for native "Save As" on Chrome/Edge. Falls back to anchor download on Firefox/Safari.
